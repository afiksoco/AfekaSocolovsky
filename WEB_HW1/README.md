
# AfekaSocolovsky - Web Development Course

## Exercise 1: Building a Website (5 points)

### Project Description
A simple website showcasing "Top 5 Beautiful Places I Want to Visit" with an interactive map interface.

### Website Structure
```
AfekaSocolovsky/
├── index.html              # Home page with interactive map
├── santorini.html          # Santorini, Greece page
├── kyoto.html              # Kyoto, Japan page
├── patagonia.html          # Patagonia, Argentina page
├── iceland.html            # Iceland page
├── bali.html               # Bali, Indonesia page
├── styles.css              # Stylesheet for all pages
├── images/
│   ├── world-map.png       # World map for home page
│   ├── santorini/          # 2 images (1.jpg, 2.jpg)
│   ├── kyoto/              # 2 images (1.jpg, 2.jpg)
│   ├── patagonia/          # 2 images (1.jpg, 2.jpg)
│   ├── iceland/            # 2 images (1.jpg, 2.jpg)
│   ├── bali/               # 2 images (1.jpg, 2.jpg)
│   └── IMAGE_INSTRUCTIONS.md
└── README.md               # This file
```

### Features
- **Home page** with interactive world map and clickable pins
- **5 location pages** with banner images and photo galleries
- **Table layout** for content organization
- **Back to Home** link on all location pages
- **Simple CSS** styling throughout
- **No external libraries** - only HTML and CSS
